Available Down Mixes
--------------------

HandBrake supports the following downmixes:

-   Mono

-   Mono (Left Only)

-   Mono (Right Only)

-   Stereo

-   Dolby Surround

-   Dobly Pro Logic II

-   5.1 Channels

-   6.1 Channels

-   7.1 Channels

-   7.1 Channels (5F/2R/LFE)



Upmixing
--------

HandBrake does not currently support upmixing audio.
